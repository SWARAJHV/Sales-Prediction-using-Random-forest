# Advertising Sales Prediction

This project demonstrates how to develop and evaluate machine learning models to predict advertising sales based on various advertising channels. The project includes data preprocessing, model training, and evaluation using Linear Regression and Random Forest models. Additionally, hyperparameter tuning is performed using Grid Search.
